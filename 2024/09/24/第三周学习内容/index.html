为什么这些类型都要大写，看得心累

数值类型

类型 大小 用途
TINYINT 1 Bytes 小整数值
SMALLINT 2 Bytes 大整数值
MEDIUMINT 3 Bytes 大整数值
INT或INTEGER 4 Bytes 极大整数值
BIGINT 8 Bytes 极大整数值
FLOAT 1 Bytes 单精度浮点数值
DOUBLE 8 Bytes 双精度浮点数值
DECIMAL 对DECIMAL(M,D) ，如果M>D，为M+2否则为D+2 小数值

日期和时间类型

类型 大小 格式
DATE 3 bytes YYYY-MM-DD
TIME 3 bytes HH:MM:SS
DATETIME 1 bytes YYYY-MM-DD hh:mm:ss
TIMESTAMP 4 bytes YYYY-MM-DD hh:mm:ss

TIMESTAMP和DATETIME的区别：

存储范围：

  • DATETIME：范围从1000-01-01 00:00:00到9999-12-31 23:59:59，适用于需要很大时间范围的应用。
  • TIMESTAMP：范围从1970-01-01 00:00:01 UTC到2038-01-19 03:14:07 UTC，主要用于记录时间戳。

时区处理：

  • DATETIME：不受时区影响，存储的日期和时间是固定的。
  • TIMESTAMP：存储为UTC（协调世界时），并在检索时根据服务器的时区进行转换，这使得它更适合记录事件的确切发生时间，特别是在跨时区的应用中。

存储方式：

  • DATETIME：占用8个字节。

BLOB 二进制形式的长文本数据

MEDIUMBLOB 二进制形式的中等长度文本数据
MEDIUMTEXT 中等长度文本数据
LONGBLOB 二进制形式的极大文本数据
LONGTEXT 极大文本数据
注意：
